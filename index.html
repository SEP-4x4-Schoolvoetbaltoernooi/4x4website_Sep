Azure App instellen

Eenmalig instellen. Daarna log je in met je Microsoft-account en het gedeelde bestand wordt automatisch geladen.
  1. Ga naar portal.azure.com en log in.
  2. Azure Active Directory → App-registraties → Nieuwe registratie. Geef een naam (bijv. "SEP Scorebord") en klik Registreren.
  3. Kopieer de Application (client) ID en eventueel de Directory (tenant) ID (of gebruik common).
  4. API-machtigingen → Toevoegen → Microsoft Graph → Gedelegeerd: voeg Files.Read.All toe. Klik daarna op "Beheerdersmachtiging verlenen".
  5. Verificatie → Platform toevoegen → Toepassing met één pagina (SPA). Vul als Redirect URI de URL van dit bestand in (of http://localhost bij lokaal gebruik).
